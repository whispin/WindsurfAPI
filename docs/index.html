接入方式

所有 SDK 直连

A Claude Code

ANTHROPIC_BASE_URL=http://IP:3003
ANTHROPIC_API_KEY=sk-your-key
claude -p "你好"

O OpenAI SDK

from openai import OpenAI
client = OpenAI(
  base_url="http://IP:3003/v1",
  api_key="sk-your-key"
)

C Cursor / Cline

Base URL: http://IP:3003/v1
API Key:  sk-your-key

部署

三条命令 上线

~/WindsurfAPI 
$ git clone https://github.com/dwgx/WindsurfAPI.git && cd WindsurfAPI
$ bash setup.sh          # 建目录 · 配权限 · 生成 .env
$ node src/index.js      # 启动服务

# Docker 部署
$ docker compose up -d --build

# 打开 Dashboard
# http://你的IP:3003/dashboard
